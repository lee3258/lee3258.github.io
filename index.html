java进阶

java进阶

1.类和对象

类指的是具有特定成员变量和操作的对象集合;

对象是具有类特性的个体;

人是一个类,具有某些特征;小明是一个人,他算是一个对象;

 

2.类的特性

类中有:

  1. 成员变量

  2. 成员函数

     

    举个类的例子:

public class human()    //类
{
    height = 172;  //成员变量
    weight = 65;   //成员变量
    age    = 22;    //成员变量
    
    void dohomework()   //成员函数
    {
        System.out.println("i want to play game!");
    }
    
    void wathtv()  //成员函数
    {
        system.out.println("hahahaha");
    }
    
    public static void main()
    {   human lee = new human();
        System.out.println(lee.height);
        System.out.println(lee.weight);
     lee.watchtv();     //成员函数和性质可以直接用
        
    }
}

 

 

构造函数: new一个对象时自动执行的函数,函数名为类名字

函数重载:多个函数同名, 这些函数是重载关系,选择函数由输入参数类型决定

        this.price = price;
        // 当函数接口与类属性名字相同时,用this.price来代替类属性
